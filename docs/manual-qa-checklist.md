# Manual QA Checklist

내부 배포 전 실제 브라우저에서 빠르게 검증할 핵심 플로우입니다.

## 1. 스튜디오 기본 생성

- `/studio` 접속 후 기본 provider가 정상 선택되어 있는지 확인
- 템플릿 드롭다운이 보이고 템플릿 변경이 가능한지 확인
- 생성 범위 옵션이 아래 4개로 보이는지 확인
  - `01~16 전체 생성`
  - `09 지문 확정까지 생성 후 검토`
  - `14 콘텐츠 생성까지 완료 후 검토`
  - `지문 검토 후 콘텐츠도 검토`
- `채팅 모드`, `파이프라인 모드` 전환이 정상 동작하는지 확인

## 2. 지문 검토 루프

- 생성 범위를 `09 지문 확정까지 생성 후 검토`로 설정
- 생성 후 지문 검토 패널이 나타나는지 확인
- 제목/지문 본문을 직접 수정할 수 있는지 확인
- `이 지문으로 다음 단계 진행` 클릭 시 10~16 단계가 이어서 실행되는지 확인
- `수정 요청 반영해 지문 다시 생성` 클릭 시 지문 단계부터 다시 생성되는지 확인
- `처음부터 완전 다시 생성` 클릭 시 전체 입력 기준으로 다시 생성되는지 확인

## 3. 콘텐츠 검토 루프

- 생성 범위를 `14 콘텐츠 생성까지 완료 후 검토` 또는 `지문 검토 후 콘텐츠도 검토`로 설정
- 독해/어휘/문법/쓰기/평가지 카드가 모두 보이는지 확인
- 각 카드에 수정 요청 입력창이 보이는지 확인
- 특정 섹션만 재생성했을 때 해당 섹션만 갱신되는지 확인
- `콘텐츠 전체 다시 생성` 클릭 시 10~14 구간이 다시 생성되는지 확인
- `이 상태로 QA/발행 진행` 클릭 시 15~16 단계가 이어서 실행되는지 확인

## 4. 파이프라인 진행 표시

- `파이프라인 모드`에서 10~14 병렬 생성 구간이 개별 step으로 보이는지 확인
- 09에서 멈춘 것처럼 보이지 않고, 10~14 각 에이전트 상태가 `running/done/error`로 갱신되는지 확인

## 5. 템플릿 선택과 미리보기

- 서로 다른 템플릿 2개를 선택해 미리보기 섹션 구성이 달라지는지 확인
- 템플릿에 따라 숨겨진 섹션이 미리보기에서 실제로 사라지는지 확인
- PDF/DOCX 파일명에 템플릿 ID가 포함되는지 확인

## 6. 저장 플로우

- 생성된 레슨을 저장할 때 저장 모달에 현재 선택된 템플릿명이 보이는지 확인
- `초안 저장`, `검토 요청`이 각각 정상 저장되는지 확인
- `검토 요청` 시 추천 검토자 자동 배정 또는 수동 선택이 정상 저장되는지 확인

## 7. 자료실 상세/내보내기

- `/library`에서 저장한 레슨을 열었을 때 상세에 템플릿명이 보이는지 확인
- 상세의 export 버튼으로 학생용 PDF/DOCX가 정상 내려받아지는지 확인
- 교사용 export 권한이 있는 계정에서 교사용 PDF/DOCX가 보이는지 확인
- 저장 시 선택했던 템플릿이 export에도 그대로 반영되는지 확인

## 8. 협업 플로우

- 검토자 계정에서 `in_review` 레슨이 `내 검토함` 또는 `/library?scope=review`에 보이는지 확인
- 카드 인라인 `승인`, `수정 요청`이 동작하는지 확인
- 코멘트 작성 후 활동 이력과 코멘트 목록이 즉시 갱신되는지 확인
- 재배정 후 작업함/자료실 배지와 링크가 즉시 갱신되는지 확인

## 9. 권한 점검

- 일반 강사 계정에서 관리자 전용 기능이 보이지 않는지 확인
- `템플릿 관리` 메뉴가 관리자/수석 강사에게만 보이는지 확인
- 일반 강사가 시스템 설정 API를 호출해도 거부되는지 확인

## 10. 회귀 확인

- `/work` 배지가 저장/검토/재배정 직후 갱신되는지 확인
- 새로고침 없이 탭 간 동기화가 되는지 확인
- 이전에 저장한 레슨 중 템플릿 정보가 없는 항목도 기본 export가 가능한지 확인
- 빌드 확인: `npm run build`

## Known Risk Notes

- 템플릿 편집기는 현재 MVP 수준입니다.
  - 블록 타입/섹션/강조색 중심 관리
  - 드래그 기반 자유 배치 편집기는 아직 아님
- 기존 레슨 데이터에는 `documentTemplate`가 없을 수 있습니다.
  - 이 경우 기본 템플릿으로 export 됩니다.
